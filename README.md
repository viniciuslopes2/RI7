Para executar o programa basta ter o Node.js (No windows é ir no site do Node.js e selecionar o instalador do Node.js de acordo com o seu computador e inclusive o seu sistema operacional). Feito isso, é só baixar o arquivo e depois rodar ele usando o node ./dist/index.js e verá a mágica, caso você faça alguma alteração no index.ts, por favor apague a pasta dist e depois na pasta mãe você abre o terminal e rode o npx tsc e aí logo em seguida colocar no mesmo terminal o node ./dist/index.js que isso fará com que o programa rode!
